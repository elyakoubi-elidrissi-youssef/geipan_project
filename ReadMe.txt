Lancer les 2 projets
normalement les nodes modules seront envoyés dans le projet mais vaut mieux s'assurer qu'ils sont là,donc il faut faire :
"npm install" sur les deux projets 
après il faut démarrer moongo dans le restgeipan avec "mongod.exe"
et démarrer "node server" aussi dans le restgeipan
et après lancer "ng serve" sur le menugeipan (si le projet ne se lance pas directement dans le navigateur, veuillez prendre l'adresse du lien depuis la console)
